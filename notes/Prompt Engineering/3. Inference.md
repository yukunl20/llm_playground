#### The Inferring Task
In LLMs, "inferring" refers to tasks where the model takes text as input and performs analysis, such as extracting labels, names, or sentiments.

Advantages Over Traditional ML

- **Speed:** Traditional machine learning requires collecting labeled data, training a model, and deploying it. With LLMs, you can start generating results immediately with just a prompt.
- **Versatility:** One model and one API can perform many different tasks (sentiment, name extraction, etc.) rather than needing separate models for each.
- **Zero-Shot Learning:** Models can often identify topics or categories without any prior training data specific to that task.

#### Sentiment and Emotion Analysis

LLMs are highly effective at identifying the underlying tone of a text.

Basic Sentiment Classification
```python
prompt = f"""
What is the sentiment of the following product review, 
which is delimited with triple backticks?

Review text: '''{lamp_review}'''
"""
```

#### Concise Outputs for Post-Processing
To make the response easier for code to process, you can instruct the model to provide a single-word answer.

```python
prompt = f"""
What is the sentiment of the following product review, 
which is delimited with triple backticks?

Give your answer as a single word, either "positive" \
or "negative".

Review text: '''{lamp_review}'''
"""
```

#### Information Extraction

You can use LLMs to extract specific details like brand names or product types and format them for easy use in Python.

Extracting Multiple Items in JSON

Formatting output as a **JSON object** allows you to easily load the data into a Python dictionary for further processing.

``` python
prompt = f"""
Identify the following items from the review text: 
- Item purchased
- Company that made the item

The review is delimited with triple backticks. \
Format your response as a JSON object with \
"Item" and "Brand" as the keys. 
If the information isn't present, use "unknown" \
as the value.

Review text: '''{lamp_review}'''
"""
```

#### Topic Identification

LLMs can summarize the primary topics of a long text or determine if a specific topic of interest is mentioned.

Zero-Shot Topic Tracking

You can provide a list of specific topics and ask the model to flag which ones appear in a text, which is useful for creating automated news alerts.

``` python
topic_list = ["nasa", "local government", "engineering", "employee satisfaction", "federal government"]

prompt = f"""
Determine whether each item in the following list of \
topics is a topic in the text below, which is delimited \
with triple backticks.

Give your answer as a list with 0 or 1 for each topic.

List of topics: {", ".join(topic_list)}

Text: '''{news_article}'''
"""
```

**Production Tip:** For production systems, it is more robust to request JSON output rather than a list, as the formatting of lists can sometimes be inconsistent.
