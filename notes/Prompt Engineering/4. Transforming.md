#### Technical Setup

Transformation tasks use the standard OpenAI completion helper function.

```python
import openai

def get_completion(prompt, model="gpt-3.5-turbo"):
    messages = [{"role": "user", "content": prompt}]
    response = openai.ChatCompletion.create(
        model=model,
        messages=messages,
        temperature=0,
    )
    return response.choices.message["content"]
```

#### Transformation Capabilities

Large Language Models (LLMs) are highly proficient at converting input text into different formats, languages, or tones.

1. Translation

LLMs can translate between hundreds of languages and even handle variations in tone (formal vs. informal).

```python
# Multiple translations in one prompt
prompt = f"""
Translate the following text to French, Spanish, and English pirate: 
'I want to order a basketball'
"""

# Specifying tone in translation
prompt = f"""
Translate the following text to Spanish in both formal and informal forms:
'Would you like to order a pillow?'
"""
```

2. Universal Translator

You can build a "universal translator" by first asking the model to identify the source language and then translating it into a target language.

``` python
prompt = f"""
Identify the language of the following text:
'Combien coûte le lampadaire'
"""
```

3. Tone Transformation

The model can rewrite text to suit a specific audience, such as converting casual slang into a professional business letter.

``` python
prompt = f"""
Translate the following from slang to a business letter:
'Dude, this is Joe, check out this spec on the standing lamp.'
"""
```

4. Format Conversion

LLMs can seamlessly convert between data formats like JSON, HTML, XML, and Markdown.

``` python
# Converting JSON to an HTML table
prompt = f"""
Translate the following Python dictionary from JSON to an HTML table 
with column headers and a title: {json_data}
"""
```

5. Spell Check and Grammar

The model can proofread text, correct grammatical errors, and even adjust the style to follow specific guidelines (like APA style).

``` python
# Basic proofreading
prompt = f"""
Proofread and correct the following text. 
If you find no errors, just say 'No errors found':
'''{text_to_check}'''
"""

# Advanced proofreading for a specific audience
prompt = f"""
Proofread and correct this review, make it more compelling, 
ensure it follows APA style, and targets an advanced reader. 
Output in markdown format.
"""
```

6. Tracking Changes (Diffs)

In Python, you can use packages like `redlines` to visualize the specific changes the model made during the transformation process.

``` python
from redlines import Redlines

diff = Redlines(original_text, model_output)
display(HTML(diff.output_markdown))
```
